The Story Behind the Code

I'm a Quality Assurance Specialist and Tech Support Manager who decided that understanding software wasn't enough — I wanted to build it.

My background in QA means I approach development with a precision-first mindset. I test before I ship, think about edge cases, and care deeply about the user experience at every level.

Currently levelling up through Code Institute's Full-Stack programme, while simultaneously building Hendoshi Store — a real fashion e-commerce business I'm taking from portfolio project to live product.

I moved from Brazil to Ireland in 2013. That kind of leap takes ambition and adaptability — the same qualities I bring to every line of code I write.
