## Overview

Personal portfolio site for **Danni Kadoshi** — full-stack developer based in Co. Wexford, Ireland. Built to showcase real shipped projects, technical range, and the story behind the work.

No template. No page builder. Custom coded from scratch.

---

## Features

- **Theme switcher** — Dark · Light · Matrix · Amber — persisted in localStorage
- **Animated hero** — staggered fade-in reveals with accent text
- **Live ticker** — scrolling activity strip
- **Timeline** — visual career and project milestones
- **Projects grid** — featured flagship card + regular project cards
- **Tech stack filter** — filter tools by category (Languages, Frameworks, Tools)
- **Scroll animations** — IntersectionObserver fade-in throughout
- **Fully responsive** — mobile hamburger menu, fluid layouts
